# Запрос данных у заказчика

Нужно заполнить и вернуть данные для включения оплаты и контактов:

1. Реквизиты для оплаты по карте (номер карты, получатель, банк).
2. Номер телефона менеджера для связи при спорной оплате.
3. ID группового чата менеджеров в Telegram (где подтверждают оплату).
4. QR-код СБП (файл/ссылка/текст для генерации).

Примечание: сейчас используются временные значения в `frontend/src/config/payment.ts` и `backend/src/config/index.ts`.
